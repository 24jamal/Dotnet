Employee Management System
## Overview
The Employee Management System is a web application built using ASP.NET MVC. It provides a platform for managing employee records, with functionalities such as user authentication, CRUD (Create, Read, Update, Delete) operations, search, pagination, and data export. This application is designed to streamline employee management processes, making it easier for businesses to maintain and update employee information.

Features
User Authentication:

Login: Secure access to the system with username and password.
Register: Allows new users to create an account.
Forgot Password: Provides a way for users to reset their password via email.
Change Password: Enables users to update their password securely.
CRUD Operations:

Create: Add new employee records.
Read (Index View): View a list of employees with search and pagination features.
Update: Edit existing employee records.
Delete: Remove employee records from the system.
Additional Features:

Search Functionality: Quickly find specific employees by name or other attributes.
Pagination: Navigate through large lists of employees with ease.
Export to CSV: Export employee data to a CSV file for external use.
Project Structure
Models: Defines the data structures for employees and user accounts.
Views: Contains Razor view files for rendering the user interface.
Controllers: Manages the logic for processing user input and interacting with the database.
Data Access Layer (DAL): Implements database operations using ADO.NET.
